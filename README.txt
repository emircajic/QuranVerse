QuranVerse is a plugin that makes easy to insert Quranic text into WordPress posts. 

== Description ==

This plugin makes it easy to cite Quranic verses (in Arabic) together with 
translations. It is quite flexible on how you can display the text, since it
uses twig templating engine, and also makes it easy to add your own templates, 
so you can integrate it with your site seamlesly.

Please note that when the plugin reaches its version 1.0.0 this means that you 
can use it on production sites. Otherwise, you are free to test it, suggest features, and play with it anyway you like. As of now, it is still in early stages, only interesting to developers wishing to contribute.


== Installation ==

1. Upload `quranverse.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to `QuranVerse Settings` in your `Settings` menu.
2. Choose the appropriate settings.
